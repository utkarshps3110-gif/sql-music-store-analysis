# Music Store Sales Analysis Using SQL

## Project Overview
This project analyzes sales data from a digital music store using SQL.
The goal is to identify revenue trends, customer purchasing patterns,
and product performance.

## Tools Used
SQL  
SQLiteStudio

## Skills Demonstrated
SELECT  
WHERE  
GROUP BY  
ORDER BY  
CASE statements

## Dataset
Chinook Music Store Database
