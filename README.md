# Music Store Sales Analysis Using SQL

This project analyzes sales data from a digital music store using SQL.  
The analysis explores customer behavior, revenue distribution, and sales trends.

## Skills Demonstrated
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- CASE statements
